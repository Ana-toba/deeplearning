# Computer Vision Inference System

Este repositorio contiene los componentes principales para la captura, procesamiento y análisis de imágenes utilizando modelos de inteligencia artificial.

## Estructura del proyecto

```text
.
├── models/
├── config.py
├── main_video2.py
├── postprocess.py
└── utils.py
```

## Descripción de archivos

- **main_video2.py**: Script principal para la ejecución del sistema.
- **config.py**: Parámetros de configuración.
- **utils.py**: Funciones auxiliares.
- **postprocess.py**: Funciones de postprocesamiento de resultados.
- **models/**: Modelos utilizados por el sistema.

## Instalación

```bash
pip install -r requirements.txt
```

## Ejecución

```bash
python main_video2.py
```
